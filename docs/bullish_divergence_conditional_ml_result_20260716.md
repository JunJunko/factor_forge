# 底背离条件 Alpha：M0–M7 Walk-forward OOF 消融结果

## 结论

本轮实验支持“底背离事件内部存在可学习的条件排序”，但不支持直接按 40bp 总交易成本上线。

- PIT 概念版本样本中，Ridge M7 的 OOF Rank IC 为 `0.0586`，Top10 相对全部背离事件的 10 日行业超额提升 `51.5bp`，4/4 个测试折为正。
- Ridge M7 的上述结果高于 20/20 次日期内标签置换；Top10-minus-all 的 placebo 均值为 `6.5bp`、最大值为 `40.4bp`。
- Ridge M7 的 Top10 行业超额毛收益为 `25.3bp`。按 20bp 总成本为 `+5.3bp`，按 40bp 总成本为 `-14.7bp`，按 60bp 总成本为 `-34.7bp`。
- M7 相对 M6、M6 相对 M1 的增量置信区间仍跨零。因此可以确认组合模型具有排序能力，但暂时不能确认交互项或全部上下文模块具有独立、稳定的增量贡献。
- 浅层 LightGBM 在 PIT 样本中明显不如 Ridge，且 M6/M7 没有稳定超过 placebo，说明当前短样本和特征维度下非线性模型更容易过拟合。

判定：**研究 Gate 通过，部署 Gate 未通过。**

## 消融定义

| 模型 | 特征块 |
|---|---|
| M0 | X：规模、流动性、波动、Beta、近期收益等控制变量 |
| M1 | X + D + T：底背离结构、振荡器改善、触碰/支撑特征 |
| M2 | X + R + C + M：市场 regime、行业/概念轮动、个股动量，不使用背离内部特征 |
| M3 | M1 + R |
| M4 | M1 + C |
| M5 | M1 + M |
| M6 | M1 + R + C + M |
| M7 | M6 + 预注册交互项 |

标签为信号后 `T+1 open → T+11 open` 的 10 日申万一级行业超额收益。训练样本按交易日等权；训练集只允许使用 `label_available_date < test_start` 的事件。所有模型在同一 scope 内使用相同事件、相同 OOF 折和相同标签。

## 样本和验证

### 长样本：行业上下文

- 数据集：81,764 个事件，2021-01 至 2026-07。
- OOF：69,482 个事件、1,052 个交易日。
- 5 个扩展窗口：依次测试 2022、2023、2024、2025、2026 年截至 7 月。
- C 只包含当时点行业轮动，不包含概念数据。

长样本最好的 Top10-minus-all 是：

| 模型 | 算法 | Rank IC | Top10-minus-all | 40bp 后 Top10 |
|---|---:|---:|---:|---:|
| M2 | Ridge | 0.0281 | 26.7bp | -42.3bp |
| M2 | LightGBM | 0.0196 | 23.5bp | -45.4bp |
| M6 | Ridge | 0.0258 | 15.7bp | -53.2bp |
| M6 | LightGBM | 0.0234 | 18.3bp | -50.6bp |

M6 没有显著优于 M1，且 Ridge M6 显著弱于 M2。长样本更像是“背离事件出现后，由市场/行业/动量上下文决定相对排序”，把全部背离内部变量和上下文同时放入并未产生稳定增益。

### PIT 概念版本样本

- 数据集：16,192 个事件，2024-12 至 2026-07。
- OOF：15,802 个事件、238 个交易日。
- 4 个扩展窗口，测试区间依次为 2025Q3、2025Q4、2026Q1、2026Q2 至 7 月。
- 概念成员使用历史快照；只聚合当日轮动候选概念的真实历史成分股。
- 只有 8.66% 的背离事件属于当日活跃候选概念。因此当前概念变量是“强轮动概念暴露”，不是完整概念画像。

Ridge 的 M0–M7 结果：

| 模型 | Rank IC | Top10-minus-all | 40bp 后 Top10 |
|---|---:|---:|---:|
| M0 | 0.0089 | -9.2bp | -75.5bp |
| M1 | 0.0514 | 35.7bp | -30.5bp |
| M2 | 0.0245 | 29.3bp | -36.9bp |
| M3 | 0.0503 | 38.0bp | -28.2bp |
| M4 | 0.0593 | 42.4bp | -23.8bp |
| M5 | 0.0382 | 28.7bp | -37.5bp |
| M6 | 0.0507 | 45.3bp | -20.9bp |
| M7 | 0.0586 | 51.5bp | -14.7bp |

关键配对消融：

| 对比 | Top10-minus-all 增量 | 95% block-bootstrap CI | 判断 |
|---|---:|---:|---|
| M1 − M0 | +45.0bp | `[+16.7, +78.1]bp` | D/T 对事件内部排序有贡献 |
| M2 − M0 | +38.5bp | `[+4.9, +70.5]bp` | 上下文在背离事件池内也能排序 |
| M4 − M1 | +6.7bp | `[-15.0, +30.0]bp` | C 的独立增量未确认 |

## Placebo 与稳定性

日期内标签置换保留每个交易日的标签分布和日期效应，只打乱股票与标签的对应关系。

Ridge M7：

- Rank IC：真实 `0.0586`；20 次 placebo 均值 `0.0204`，最大 `0.0503`。
- Top10-minus-all：真实 `51.5bp`；placebo 均值 `6.5bp`，最大 `40.4bp`。
- Top-minus-bottom：真实 `76.8bp`；placebo 均值 `15.5bp`，最大 `56.7bp`。
- 经验分位数均为 `1.00`，即真实结果高于 20 次置换。
- 四个测试折的 Top10-minus-all 分别为 `22.2bp`、`75.0bp`、`38.3bp`、`73.5bp`，4/4 为正。
- 40bp 成本后的 Top10 只有 1/4 折为正。

LightGBM M6/M7 的结果没有可靠超过 placebo，暂不应进入调参阶段。

## 因子构造检验

当前结果与最初实验设想基本一致，但揭示了三个限制：

1. 底背离内部特征不是独立策略，而是事件池内的重要排序信息。PIT 样本中 M1 显著优于 M0，支持继续使用 D/T。
2. 上下文因子确实能排序，但和 D/T 存在较强冗余。R、C、M 单独加入 M1 的增量大多不显著，M6 也没有显著优于 M1。
3. 概念因子覆盖过窄。当前只测试“当日 Top20 轮动概念的成分股暴露”，事件覆盖率 8.66%，不能据此否定完整概念画像的价值。

Ridge M7 的主要权重来自背离下降结构、MACD 改善、均线距离、换手变化、市场宽度和行业轮动。交互项中权重较大的包括：

- 触碰支撑次数 × 个股动量加速度；
- 成交激活 × 行业宽度变化；
- 背离评分 × 市场回撤；
- 个股相对动量 × 行业动量。

这些权重只用于形成下一轮假设，不能作为已确认的经济机制；M7 − M6 的区间仍跨零。

## Gate

### 研究 Gate

- OOF Rank IC 置信区间下界大于 0：通过。
- Top10-minus-all 置信区间下界大于 0：通过。
- 4/4 测试折 Top10-minus-all 为正：通过。
- 高于 20 次日期内标签置换：通过。

### 部署 Gate

- 40bp 成本后 Top10 均值大于 0：未通过。
- 多数测试折 40bp 后为正：未通过，只有 1/4。
- M6 相对 M1、M7 相对 M6 的独立增量显著：未通过。
- Top20 仍保留 alpha：未通过。

## 下一轮实验建议

1. 扩展完整 PIT 概念画像，不只使用 Top20 轮动候选概念；同时保留“活跃概念暴露”作为单独子块。
2. 把 M1、M4、M6、M7 作为候选，不调 LightGBM；对 Ridge/ElasticNet 做嵌套 walk-forward 正则强度选择。
3. 增加成本感知标签：预测毛超额、成功概率和预期可成交成本，最终按预期净 alpha 排序。
4. 对 Top5/Top10 做真实重叠持仓组合回测，加入停牌、涨跌停、容量、换手、行业暴露和 10 日持有期资金占用。
5. 将 25bp 作为当前成本上限；若真实执行成本高于该水平，本信号只能作为其他策略的过滤器或减仓/加仓条件。
