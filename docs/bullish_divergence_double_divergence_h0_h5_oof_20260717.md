# 连续双背离结构因子 H0–H5 Walk-forward OOF 结果

日期：2026-07-17

## 1. 本轮重建口径

结构因子使用三个因果低点 `P → A → B`：

- 第一段 `P → A` 已满足底背离；
- 第二段 `A → B` 也满足底背离；
- 两段均要求低点间隔、价格几何、中间反弹、进入低点前下跌，以及 RSI14、MACD/ATR、下跌速度至少一个改善；
- `structure__double_divergence_present` 表示连续双背离；
- 第一段、第二段的价格、RSI、MACD、速度、量能、反弹和一致性均保留为原子特征；
- 第二段相对第一段的变化作为 trend 特征；
- `structure__double_divergence_trend_score` 是 trend 原子的截面百分位复合分数。

`structure__pivot_p_date` 仅作诊断，已明确排除在模型输入之外。

## 2. 数据门禁

- 新版 v2 事件：83,099，与重建前完全一致，说明 S 的替换没有改变 D/T 事件样本。
- OOF 成熟样本：81,764，事件键 100% 一一匹配。
- 连续双背离：9,176，占全部 OOF 事件 11.22%，占结构字段非缺失事件 11.38%。
- 覆盖：1,090 个交易日、4,087 只股票。
- 趋势分数中位数 51.41，10%/90% 分位为 30.18/72.41，具备足够离散度。

## 3. OOF 设计

特征块：

- `X`：规模、流动性、波动、beta、收益和回撤等控制变量；
- `D`：本次底背离原子与 v2 分数；
- `T`：当前背离位置是否被 K 线触碰、触碰距离、次数、接受度等；
- `S`：本轮连续双背离结构；
- `R/C/M`：市场 regime、行业或 PIT 概念轮动、个股动量。

消融臂：

- H0 = X
- H1 = X + D + T
- H2 = X + S
- H3 = X + D + T + S
- H4 = X + D + T + R + C + M
- H5 = X + D + T + S + R + C + M

标签为未来 10 个交易日行业超额收益。采用 expanding walk-forward、标签可得日 embargo、完全 OOF 预测。模型为 Ridge 与 LightGBM。主组合统计采用 Top10、40 bps 成本；成对增量中的 `top_minus_all` 不受统一成本平移影响。

long 场景含 81,764 个事件、2022-01-01 至 2026-07-10 五个测试 fold。PIT 概念场景含 16,192 个事件、2025-07-01 至 2026-07-10 四个季度 fold。

## 4. S 的成对增量结果

下表的收益增量为日均 `top_minus_all`，单位为小数收益率；括号为日度 block bootstrap 95% 区间。

### Long / 行业上下文

| 模型 | 对比 | Δ Rank IC | Δ Top10−All |
|---|---|---:|---:|
| Ridge | H2−H0 | -0.0007 `[-0.0064, 0.0048]` | -0.00048 `[-0.00115, 0.00022]` |
| Ridge | H3−H1 | +0.0005 `[-0.0051, 0.0064]` | -0.00048 `[-0.00116, 0.00014]` |
| Ridge | H5−H4 | +0.0004 `[-0.0050, 0.0048]` | +0.00014 `[-0.00046, 0.00073]` |
| LightGBM | H2−H0 | -0.0010 `[-0.0088, 0.0074]` | +0.00018 `[-0.00079, 0.00105]` |
| LightGBM | H3−H1 | +0.0025 `[-0.0051, 0.0122]` | -0.00016 `[-0.00090, 0.00074]` |

结论：所有区间均跨 0，S 在长样本中没有稳定增量。

## 5. 绝对组合表现

PIT 场景中：

- Ridge：H1 Rank IC 0.0514、Top10−All 0.00357；H3 Rank IC 升至 0.0581，但 Top10−All 降至 0.00286。
- Ridge：H4 Rank IC 0.0507、Top10−All 0.00453；H5 Rank IC 0.0509、Top10−All 降至 0.00397。
- LightGBM：H1 Rank IC 0.0393、Top10−All 0.00369；H3 Rank IC 0.0418、Top10−All 降至 0.00238。
- LightGBM：H4 Rank IC 0.0096、Top10−All 0.00129；H5 Rank IC 0.0128、Top10−All 降至 0.00096。

各主臂的 40 bps 成本后 Top10 绝对收益仍为负。当前结果只能说明相对事件池存在一定排序能力，不能称为可交易的 long-only 绝对 alpha。

## 6. Placebo 与稳定性

- PIT 使用 20 次交易日内标签打乱；H3 的真实 Rank IC 超过 Ridge 19/20、LightGBM 20/20 个 placebo，说明 D/T/S 联合模型学到的不是纯随机关系。
- H5 的真实 Rank IC 超过 Ridge 19/20、LightGBM 17/20 个 placebo。
- 但当前 placebo 针对 H3/H5 整个模型，不是针对 `H3−H1`、`H5−H4` 的成对增量。因此它不能把联合模型的真实性归因给 S。
- PIT LightGBM 第一个 fold 训练样本只有 206 个，产生日内常数预测并导致 Rank IC 为 NaN；PIT 非线性结果需在概念快照历史扩展后复验。

## 7. 因子含义的数据检验

连续双背离不是正向增强器，至少在当前定义和标签下不是：

- 全样本连续双背离事件的 10 日行业超额均值为 -0.372%，其他事件为 -0.131%，差值 -0.242%。
- PIT 子样本分别为 -0.943% 与 -0.377%，差值 -0.567%。
- 在连续双背离内部，趋势分数的逐日 Spearman IC：long 为 -0.0177，PIT 为 -0.0426。
- long 全局五分位的收益并不正向单调，Q1 至 Q5 分别约为 -0.233%、-0.296%、-0.444%、-0.551%、-0.338%。

这解释了为什么 S 能在简单 Ridge 中提升排序，却不能改善 Top10：模型更可能把它用作“反复探底仍未确认、下跌延续”的风险标记，或者对其取负权重，而不是给连续双背离加正权重。

Ridge 在 long 场景中对 `macd_atr_improvement_trend`、复合 trend score 和第一段 MACD 改善的平均系数为负，也与上述解释一致。原子特征仍有信息，但复合分数的预设正方向不成立。

## 8. 本轮结论与下一步

本轮不支持“连续两次底背离越强，随后正向 alpha 越高”的原始方向。它支持一个更窄的命题：连续双背离可能是风险过滤变量，在只有控制变量的 PIT 线性模型中有排序增量；该信息与 D/T、regime、轮动和动量高度重叠，尚未转化为 Top10 收益。

下一轮应停止把 27 个 S 字段整体塞入模型，改做方向识别消融：

1. `S_flag`：只保留连续双背离标志，检验是否应直接降权或排除；
2. `S_leg`：只保留第一段和第二段的原子量，检验水平信息；
3. `S_delta`：只保留第二段减第一段的趋势量，不含复合 score；
4. `S_score`：只保留复合分数，明确检验它本身；
5. `S_gate`：让 S 与 regime、概念轮动、动量做显式交互，检验“什么环境下二次背离是吸筹，什么环境下是下跌中继”；
6. 对上述每个版本运行 paired incremental placebo，即直接比较真实与打乱标签下的 `H3−H1`、`H5−H4` 分布。

优先级最高的是 `S_flag` 风险过滤与 `S_gate` 条件交互，而不是继续调高当前正向 trend score 的权重。
