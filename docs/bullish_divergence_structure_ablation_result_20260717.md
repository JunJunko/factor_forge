# A/B 低点结构迁移：H0–H5 Walk-forward OOF 实验

## 结论

本轮没有确认 A→B 低点结构迁移因子具有独立、稳定的增量 alpha。

- 结构因子单独相对控制变量存在弱排序信息，但加入原 D/T 后增量接近零。
- 加入完整市场、行业/概念和动量上下文后，全部结构项在 PIT 样本中略微退化。
- 去掉综合分数与分位数的内部共线后，`S_atomic` 在 PIT Ridge 完整模型中带来 `+6.5bp` Top10-minus-all 增量，但 95% CI 为 `[-0.4,+13.2]bp`，尚未通过增量 Gate。
- 浅层 LightGBM 加入结构因子后普遍退化，不支持通过非线性模型继续挖掘当前这组结构变量。
- 结构分数的年度方向不稳定，2025 年与其他年份反向，是当前最重要的风险。

判定：**结构描述保留为研究候选；独立增量 Gate 未通过；不进入部署。**

## 实验定义

| Arm | 特征块 | 目的 |
|---|---|---|
| H0 | X | 控制基准 |
| H1 | X + D + T | 原 M1 背离/触碰基准 |
| H2 | X + S | 结构因子单独排序能力 |
| H3 | X + D + T + S | S 相对 M1 的增量 |
| H4 | X + D + T + R + C + M | 原 M6 完整上下文基准 |
| H5 | X + D + T + S + R + C + M | S 相对完整模型的增量 |

S 包含：下跌减速、收盘位置变化、ATR 标准化低点修复、下影线变化、实体变化、改善项计数、综合分数和截面分位数。

标签、fold、purge、训练权重、Ridge/LightGBM 参数和成本假设均沿用上一轮；没有针对 S 调参。

## 构造敏感性

为了排除综合分数、分位数和原子项同时进入造成的共线，额外固定 Ridge 参数测试四种表示：

- `all`：全部结构项；
- `atomic`：只保留原子变化与改善计数；
- `score_only`：只保留综合分数；
- `rank_only`：只保留综合分数的截面分位数。

PIT Top10-minus-all 增量：

| 表示 | H3−H1 | 95% CI | H5−H4 | 95% CI |
|---|---:|---:|---:|---:|
| all | +1.82bp | `[-7.08,+11.49]` | -2.31bp | `[-10.66,+4.77]` |
| atomic | +4.94bp | `[-2.71,+13.98]` | +6.50bp | `[-0.40,+13.19]` |
| score_only | +7.67bp | `[-3.68,+18.96]` | +5.64bp | `[-0.88,+13.16]` |
| rank_only | +5.90bp | `[-3.52,+15.87]` | +0.63bp | `[-9.39,+8.46]` |

`atomic` 是完整模型中最值得保留的版本，但仍只是边缘候选。其 H5 Top10 毛行业超额约 25.6bp：20bp 成本后约 `+5.6bp`，40bp 成本后约 `-14.4bp`。

## 为什么没有形成稳定增量

### 1. 与原 D/T 高度重叠

- `structure__descent_easing` 与 `div__downside_velocity_change` 秩相关 `0.872`。
- `structure__transition_score` 与 `div_v2__q_velocity_improvement` 秩相关 `0.805`。
- 收盘位置变化、下影线变化也分别与原 close-location、lower-shadow、touch-distance 有 0.4–0.53 的相关性。

这说明当前 S 主要是把原速度改善与K线承接换了一种表达，而不是增加新的路径信息。

### 2. 时间方向不稳定

事件内结构分数 Q5−Q1 的年度行业超额：

| 年份 | Q5−Q1 |
|---:|---:|
| 2021 | +36.8bp |
| 2022 | +55.4bp |
| 2023 | +7.7bp |
| 2024 | +124.0bp |
| 2025 | -43.6bp |
| 2026 | +45.0bp |

结构改善不是无条件同方向因子，更可能依赖市场修复阶段、拥挤度或下跌类型。

### 3. 非线性模型没有解决问题

LightGBM 在 PIT 中 H3 明显弱于 H1，H5 也弱于 H4。当前样本不支持用树模型自动寻找结构阈值或交互。

## 下一条假设

不建议继续调整当前综合分数权重。更合理的下一步是：

1. 默认只保留 `S_atomic`，综合 score/rank 作为诊断，不同时进入模型。
2. 构造真正新增的“盆地形状”变量：低位停留宽度、A/B 附近收盘次数、反弹与回落持续时间不对称、下降路径曲率、低点区间的价格接受时间。
3. 预注册 `S_atomic × regime transition`，验证结构改善是否只在市场从 retreat 转为 repair、行业宽度加速改善时有效。
4. 仍以 H5−H4 的配对 OOF 区间下界大于零作为增量 Gate，不因某个年份或某次 placebo 调参。
